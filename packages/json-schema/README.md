# @targetd/json-schema

> Creates JSON schema from targetd data objects

## Installation

| JS Runtime | Command                                          |
| ---------- | ------------------------------------------------ |
| Node.js    | `npx jsr add @targetd/api @targetd/json-schema`  |
| Bun        | `bunx jsr add @targetd/api @targetd/json-schema` |
| Deno       | `deno add @targetd/api @targetd/json-schema`     |
